## מה מייחד את הסדרה

רוב המפתחים עובדים עם מודלים כמו קסם.  
קצת API, קצת פרומפטים ומקווים שהמודל יעשה את הדבר הנכון.

הסדרה נותנת לך את השפה ואת ההבנה שמאחורי הקלעים:

- איך Embeddings עובדים ולמה הם מייצגים משמעות  
- למה מרחקים במרחב וקטורי קובעים הצלחה או כישלון  
- מה Gradient Descent באמת עושה בכל צעד  
- איך ממירים הבנה אמיתית למערכות Production יציבות  
- ואיך מפתחים תוך מודעות למגבלות, הסתברויות ורעש של מודלים  

זהו הידע שמפריד בין שימוש חובבני ל יכולת הנדסית אמיתית.
